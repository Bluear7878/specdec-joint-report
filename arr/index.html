2. ARR v0.1 к°Җм„Ө¶

Frozen adapter-disabled original targetмқ„ p0, live targetмқ„ p, drafterлҘј qлқј н•ңлӢӨ. Drafterк°Җ token zлҘј м ңм•Ҳн–Ҳмқ„ л•Ң original targetкіј кіөмң н•ҳлҠ” exact proposal massмқҳ posterior responsibilityлҘј мӮ¬мҡ©н•ңлӢӨ.

omega(z) = min(1, p0(z) / q(z))
q(z) * omega(z) = min(p0(z), q(z))

proposal mass = shared mass + draft-only excess
              = min(p0,q) + [q-p0]+

q(z)>p0(z)мқј л•Ңл§Ң proposal coordinateлҘј мҳ¬лҰ° safe label rлҘј л§Ңл“ лӢӨ.

r(z) = t
r(v != z) = p0(v) * (1-t) / (1-p0(z))

p0(z) <= t <= q(z)
TV(p0,r) <= 0.01
KL(r || p0) <= 0.005

мқҙм „ proposalмқҳ frozen responsibility productлҘј S_kлқј н•ҳл©ҙ target lossлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

L_target = L_native_CE
         + sum_{i,k} S_{i,k} * omega_{i,k} * active_{i,k}
             * H(stopgrad(r_{i,k}), p_theta(.|h_{i,k})) / (B*K)

Targetм—җлҠ” positive soft-label cross-entropyл§Ң нқҗлҘҙкі  p0, q, sampling, omega, S, active, rлҠ” stop-gradientлӢӨ. Drafter lossлҠ” LK Lossesмқҳ pathwise Hybrid LKлҘј мӮ¬мҡ©н–ҲлӢӨ.

L_draft = sum_{i,k} S_live * HybridLK(stopgrad(p_theta), q_phi) / (B*K)
eta = 3, gamma = 1, K = 3

eta=3мқҖ confirmatory panelкіј л§һм¶ҳ Hybrid LK scheduleмқҙкі  target-side coefficient beta=1мқҖ ліё м—°кө¬мқҳ мӢӨн—ҳк°’мқҙлӢӨ. мғҒм„ё м •мқҳлҠ” ARR method proposal ARR_METHOD_PROPOSAL_KO.mdкіј prospective protocol ARR_VALIDATION_PROTOCOL_KO.mdм—җ кі м •н–ҲлӢӨ.
